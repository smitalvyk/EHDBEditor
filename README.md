# Event Horizon Database Editor

A web-based database editor designed specifically for modding the game Event Horizon. This tool provides a user-friendly graphical interface to view, modify, and manage game data files (JSON and XML).

## Features
- **Visual Editing:** Dedicated UI components for editing complex game structures like Weapon Slots, Ship Layouts, Behavior Trees, and Loot Tables.
- **Media Preview:** Built-in support for previewing game sprites (PNG, JPG) and listening to audio files (OGG, WAV) directly within the editor.
- **Raw Data Access:** Fallback to a raw code editor for unsupported formats or manual JSON/XML adjustments.
